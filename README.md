# aerocommand

Original game built in HTML with Gemini 3 pro, developed further with OpenAI codex.

## Project structure

- `index.html`: Markup and game bootstrapping references.
- `css/styles.css`: Styling and visual theme.
- `js/game.js`: Gameplay and runtime logic entrypoint.
- `assets/`: Reserved for future static game resources.
  - `assets/images/`
  - `assets/audio/`
  - `assets/data/`
